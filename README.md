## 1. メタ認知と思考方法
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**メタ認知:**|自己の思考プロセスを意識的に観察・分析するプロンプト。||
|  * 思考観察型:|思考の過程を意識的に追跡し、分析するプロンプト。|「メタ認知を使って、この問題を解決してください。以下のような思考パターンで対応してください。」|

## 2. 思考法拡張
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**思考法拡張:**|多様な思考アプローチを促すプロンプト。||
|  * セマンティック検索型:|意味的な関連性を重視した情報探索を促すプロンプト。|「セマンティック検索を用いて、環境問題に関する最新の相互関連データを収集し、包括的な分析を提供してください。」|
|  * 水平思考型:|創造的で非線形的な思考を促すプロンプト。|「水平思考で、従来とは全く異なるアプローチから都市交通問題の解決策を提案してください。」|
|  * 人間模倣型:|人間らしい思考や表現を模倣するプロンプト。|「人間のように、感情と論理を織り交ぜながら、人工知能の倫理について説明してください。」|
|  * 段階的複雑化型:|情報の複雑さを徐々に増加させるプロンプト。|「徐々に複雑さを増しながら、量子力学の基本概念を説明してください。初心者にも理解できるようにゆっくりと深く掘り下げていってください。」|

## 3. 推論と解析
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**推論プロセス:**|系統的な推論アプローチを要求するプロンプト。||
|  * タスク適応型:|タスクに応じた最適な推論方法を自律的に選択するプロンプト。|「タスクごとに適切な推論プロセスを自ら考えて答えてください。気候変動が医療システムに与える影響を分析する際、どのようなアプローチを取りますか？」|
|  * 要約分析型:|情報の要約と深い分析を組み合わせたプロンプト。|「まず、この気候変動に関する論文を300字以内で要約してください。次に、その要約に基づいて、医療システムへの3つの主要な影響を具体例とともに説明してください。」|

## 4. 思考プロセス
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**思考順序指定:**|特定の思考順序や方法論を指示するプロンプト。||
|  * ステップバイステップ型:|段階的な思考と解決を促すプロンプト。|「ステップバイステップで、この複雑な数学の問題を解いてください。各ステップを詳細に説明してください。」|
|  * AtoZ思考型:|アルファベット順で体系的に考えるプロンプト。|「AtoZで、イノベーションを生み出すための創造的思考プロセスを説明してください。」|
|  * ゴール指向型:|明確な目標設定と達成プロセスを重視するプロンプト。|「Goal=持続可能な都市開発 F(目標達成のためのステップバック質問)=AtoZ=∫F(ステップ)=結果」|

## 5. 出力品質管理
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**品質制御:**|出力の信頼性と正確性を管理するプロンプト。||
|  * ハルシネーション防止型:|誤った情報や架空の情報の生成を防ぐプロンプト。|「ハルシネーションを出さないで！根拠のある情報のみを提供してください。」|
|  * 不確実性対応型:|不確実な情報に対する誠実な対応を求めるプロンプト。|「分からないときや回答に自信がないときは、必ず"分からない"と答えてください。」|
|  * エラー網羅型:|可能な限り多くの誤りパターンを考慮するプロンプト。|「人のあらゆる間違いパターンを網羅するように、この技術仕様書をレビューしてください。」|

## 6. モチベーションと挑戦
|中分類 * 小分類|説明|例|
|-----|-----|-----|
|**挑戦型:**|限界を超える姿勢や競争的アプローチを促すプロンプト。||
